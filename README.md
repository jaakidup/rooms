
# Rooms

Chat room server with Token Authentication

  
**To run Rooms**

    npm install

    npm run start

Authenicate your user and copy the returned token .
Add the JWT Token in all subsequent requests to the server.
Set header "Authorization" : "Bearer {Token}"
